Привет, тут публикую lua скрипты для мода OpenComputers, которые были написаны с помощью AI. Для использования в личных целях в игре. Возможно кому то пригодятся.

1. [sshd] - SSH сервер и клиент, отправляет удаленные команды по сети. Для подключения используйте uuid модема удаленного сервера. Интерактивные команды такие как edit не работают.

Почему edit ломает SSH:
* edit <file> в OpenOS запускает полноценный текстовый редактор и ждёт ввода с терминала.
* SSH-сервер не проксирует stdin/stdout в реальном времени, а просто запускает команду и ждёт её завершения (через временный файл).
* При запуске edit, процесс зависает, потому что нет настоящего терминала для редактирования → сервер не завершает команду → клиент ждёт ответ → Timeout.
То есть любой интерактивный процесс не работает в текущей реализации SSH. Если кто знает как исправить, было бы здорово.
